# 浏览器历史记录 {#history}

导入浏览器历史记录后，在时间线页面刷新数据时，会自动拉取

::: warning 为什么有些历史记录没有显示?

该软件通过复制本地浏览器的数据库，以备份获取数据。若刚浏览网页后直接刷新获取数据，可能不会正常运行，特别是在Firefox上
:::
