# 弹弹play {#dandanplay}

[弹弹play - 全功能“本地视频+弹幕”播放器](https://www.dandanplay.com/)

通过在扩展里设置端口号和路径，使shion连上弹弹play服务

::: tip
记得在自动监听里添加弹弹play
:::
